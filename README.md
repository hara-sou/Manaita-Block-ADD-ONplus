このアドオンはjava版のManaitaやManaita Plusなどのまな板MODの要素を統合版で再現し、オリジナルアイテムを追加したアドオンとなっています。

アドオンの詳細は[wiki](https://github.com/hara-sou/Manaita-Block-ADD-ONplus/wiki)に記載していますのでそちらから確認してください

## 注意事項
* 自作発言,二次配布は行わないでください
* アドオンの改造は個人利用の範囲でのみ許可します。改造したアドオンの再配布は行わないでください
